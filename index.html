1. Women's Work

A digital composition of gridded letterforms created using CSS flexboxes and subtle hover interactions to draw attention to the similarities between traditionally feminine crafts and coding.

2. Get to Know Me

A get-to-know-me game styled to look like a desktop computer from the early 2000's created for my Next-Level Coding course. This site uses JavaScript to randomize the contents of the folders and encode the correct answers.

3. Packaging Generator

A generator that produces packaging based on data related to senses entered into a Google sheet. This code uses the basic methods of accessing an API and somewhat complex for-loops and conditionals.

4. Logo Generator

A generator that produces randomized logos for the New York Public Library that look like flowers of various colors and complexity. This code uses extensive for-loops and conditionals as well as range inputs.
